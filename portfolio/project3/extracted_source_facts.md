## 2. 5단계 백서 작성을 위한 원천 팩트 분류

### [1단계] 개요 (What & Why)
* **What (정의)**: Unity XR Interaction Toolkit 라이브러리를 기반으로 가상현실(XR) 내 물리적 상호작용 기술을 학습하고, 이를 응용하여 사격 게임 및 피규어 룸 미니게임을 제작한 프로젝트.
* **Why (R&D 동기)**: 가상 공간 내에서의 현실적인 Grab(잡기), Socket(거치), Lever(회전 기믹) 동작 구현 원리를 깊이 학습하고, 커스텀 소켓 필터링(Lock Grid Socket) 메카닉 및 Shader Graph 기반 시각 연출(Phase/Dissolve)을 적용하기 위함.

### [2단계] 시스템 구조 (Architecture)
* **상호작용 기본 요소**:
  * **Interactor**: 컨트롤러(주로 사용자의 손)에 부착되어 물리적 입력을 감지하고 상호작용을 처리하는 컴포넌트.
  * **Interactable**: 잡기, 던지기, 누르기 등 상호작용 대상이 되는 물리 가상 오브젝트.
  * **Interaction Layer**: Interactor와 Interactable 간의 상호작용 필터링 및 제어 규칙.
  * **Interaction Event**: 물체가 잡혔을 때 특정 트리거 행동을 바인딩하기 위한 커스텀 이벤트.
* **주요 컴포넌트 메카닉**:
  * **Socket Interactor**: Interactable 물체를 특정 슬롯/소켓 위치에 도킹 및 배치하는 컴포넌트 (예: 총기 케이스 내부 거치).
  * **Lever**: 회전 기믹을 지원하는 Interactable을 응용하여 총기 케이스의 개폐 문을 여닫는 상호작용 구현.
  * **Lock Grid Socket (커스텀 확장)**: 기본 Socket Interactor 기능을 확장하여, Interaction Layer 검증 외에도 삽입되는 오브젝트의 **Key 컴포넌트 내 특정 문자열 값(Key)을 직접 검사**하여 필터링하는 컴포넌트.

### [3단계] 핵심 기능 및 구현 (Implementation)
* **사격 게임**:
  * **진행 흐름**: Lever로 구현된 총기 케이스의 문을 열어 잠금을 풀고 ➡️ 케이스 내부 Socket Interactor에서 총기를 해제 ➡️ Grab Interactor로 총을 잡으면 사격 게임 시작 이벤트가 트리거됨.
* **피규어 룸**:
  * **진행 흐름**: ColorPicker를 작동하여 원하는 도색 색상을 선택 ➡️ 선택한 색상을 입힌 스프레이 캔이 생성됨 ➡️ 생성된 스프레이 캔을 쥐고 분사하여 피규어 도색 ➡️ 완성된 피규어를 진열장의 Lock Grid Socket에 거치.
* **시각 효과 (Shader Graph)**:
  * **Phase 효과**: 스프레이 캔 등 소품 생성 시 서서히 외형이 나타나는 셰이더 효과.
  * **Dissolve 효과**: 플레이어가 위치한 공간(룸)을 전환할 때 화면 및 배경이 분해되며 넘어가는 디졸브 효과.

### [4단계] 고민과 선택 (Trade-offs)
* **오브젝트 배치 필터링 방식 결정**:
  * **대안 A (기본 XRSocketInteractor)**:
    * *장점*: Unity 패키지 내장 기능으로 구현 비용이 낮고 즉시 사용 가능.
    * *단점*: Interaction Layer 단위로만 필터링이 가능하므로, 여러 종류의 피규어 중 특정 칸에 특정 피규어만 도킹되도록 하는 정밀 필터링(개별 인스턴스 검증) 구현 불가.
  * **대안 B (커스텀 Lock Grid Socket)**:
    * *장점*: 진입하는 오브젝트의 Key 컴포넌트 검사(`KeyValue` 문자열 일치 검사) 단계를 추가하여, 특정 피규어를 올바른 진열 위치에만 정확히 거치할 수 있는 구조적 격리 성공.
    * *단점*: 모든 진열 대상 오브젝트에 고유 Key 스크립트를 관리하고 할당해야 하는 데이터 관리 리소스가 발생함.
    * *결정 Rationale*: 피규어 룸 진열 메카닉의 오동작 방지 및 엄격한 퍼즐 기믹 유효성 검증을 위해 대안 B 채택.

### [5단계] 결과 및 배운 점 (Retrospective)
* **성과**: XR Interaction Toolkit의 물리 컴포넌트(Grab, Socket, Lever) 및 Interaction Layer 커스텀 연동을 완료하고, Shader Graph 연동 시각 효과(Phase, Dissolve)와 Lock Grid Socket을 사용한 피규어 조립형 진열 메카닉의 유효성 검증 완료.
* **한계**: 18일간의 타이트한 단기 개발 일정 제약으로 인해 다양한 미니게임 콘텐츠 수 확장 부족.
* **계획**: 차후 룸 전환(Dissolve) 메카닉과 연동한 실시간 멀티플레이 네트워킹을 도입하여 여러 사용자가 협동으로 피규어를 도색하고 조립하는 메카닉 확장 구현 예정.
