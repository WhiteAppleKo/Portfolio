1. XR Interaction Toolkit 물리 설계

가상 공간 내 물체와의 상호작용 신뢰성을 제어하기 위해 Interactor(감지 주체), Interactable(피감지 오브젝트), Interaction Layer(필터링 시스템) 설계를 표준화했습니다.

  • Interaction Layer Mask: 동일한 물리/기능 레이아웃에 소속된 장치들만 결합(잡기, 장착)되도록 1차 필터링 처리하여 가상 현실 조작 오류를 예방했습니다.
  • Interaction Event (Activate): 컨트롤러의 트리거 누름을 런타임 이벤트로 감지하여 사격 격발, 스프레이 분사 등의 기믹을 물리 모듈과 비동기식으로 체이닝했습니다.

2. Lock Grid Socket (커스텀 2중 보안 소켓)

기존 XRSocketInteractor는 레이어 검증만 거치기 때문에, 진열대 칸마다 아무 피규어나 결합되는 한계가 있었습니다. 이를 해결하기 위해 결합 대상 오브젝트의 고유 식별 키값(Key Data)을 2차 검증하여 고유 슬롯에만 안전 체결(Snap)되도록 맞춤 상속 확장했습니다.

// XRSocketInteractor 확장 기반의 2중 필터 검증
public override bool CanSelect(IXRSelectInteractable interactable)
{
    // 1. 부모 레이어 마스크 비교 통과 여부 검사
    bool bBaseCanSelect = base.CanSelect(interactable);
    if (!bBaseCanSelect) return false;

    // 2. 피규어 컴포넌트의 고유 키 식별 데이터 대조 검증
    var figureKeyComp = interactable.transform.GetComponent<FigureKeyData>();
    if (figureKeyComp == null) return false;

    // 3. 소켓 지정값과 피규어 지정 키 코드의 1:1 일치 여부 반환
    return figureKeyComp.FigureKey == m_requiredSocketKey;
}

3. 사격 상호작용 및 물리 액티베이션

가상 권총 발사 기믹 구현을 위해 물리 레버 조작, 잡기 스태프 제어 및 발사 로직을 리액티브 기반으로 설계했습니다.

  • 레버 물리 연동: 총기 가방의 손잡이/레버(Lever) 물리 회전 조인트를 잡고 규정 각도 이상 스윙 조작 시, 기믹이 활성화되며 덮개가 개방됩니다.
  • 슈팅 기믹 시퀀스: 총을 쥐는 순간(Select Enter) 씬 내 과녁과 게임 로직이 준비되며, 트리거 입력 시(Activate) 권총 슬라이드 물리 애니메이션, 탄피 배출 물리, 오디오 소스가 일괄 비동기 구동되어 발사됩니다.

4. 스프레이 페인팅 및 텍스처 UV 맵 매핑

피규어 룸의 실시간 채색 기믹을 구현하기 위해 월드 좌표계 레이캐스트와 2D UV 좌표계를 동적 연산하는 채색 파이프라인을 구축했습니다.

  • ColorPicker 연동 스폰: UI 컬러 픽셀 좌표를 파싱해 생성한 스프레이 캔을 인스턴스화할 때, Shader Graph로 짠 Phase 효과가 순차 가동되며 출현합니다.
  • Paint in 3D UV 매핑: 스프레이 노즐 끝에서 레이케스트를 방출해 피규어 콜라이더와 부딪친 월드 3D 점을 2D UV 텍스처 픽셀 공간으로 실시간 역산 매핑하여 브러시 드로잉 페인트를 정밀 분사 채색합니다.

5. Dissolve & Phase Shader Graph 제작

XR 프로젝트 컨셉의 비주얼 극대화를 위해 노이즈를 픽셀 단위로 마스킹하여 소멸/생성을 묘사하는 셰이더를 Shader Graph로 구현했습니다.

Phase & Dissolve 렌더 마스킹 Rationale
스프레이 스폰 시 사용되는 Phase 효과는 스캔 라인을 월드 y축 좌표와 비교하여 라인이 채워지는 시각화를 구축했습니다. 룸 공간이 전환될 때 사용되는 Dissolve 효과는 Simplex Noise 맵과 시간 매개변수 값을 픽셀 셰이더의 clip() 연산과 연계하여 픽셀을 순차 기화시키고, 잘려나간 단면에 강한 이미시브 발광 밴드를 둘러 연출했습니다.
