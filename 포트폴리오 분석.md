## 3. 기술 스택
- **핵심 라이브러리**: React 18 (CDN), Babel Standalone (CDN), Tailwind CSS 3 (CDN)
- **텍스트/다이어그램 렌더링**: Marked (Markdown 파서), Mermaid 11 (다이어그램 라이브 렌더러), Highlight.js (구문 강조)
- **배포**: GitHub Pages (전체 파일 정적 배포, `.github/workflows/deploy-pages.yml`)

## 4. 주요 구성 요소 상세
### 1) 온사이트 리더 (`guide.html`)
- Fetch API 사용하여 `src/*.md` 마크다운 내용 동적 로드.
- 로컬 `file://` 프로토콜 실행 시 브라우저 CORS 정책으로 차단됨 (정적 웹 서버 실행 필요).
- 로드 후 이미지 경로 자동 치환 (`../assets/` -> `assets/`) 및 Mermaid 코드블록 라이브 렌더링 수행.

### 2) 발표 덱 (`slides/deck.html`)
- React CDN 방식의 단일 HTML 발표 자료.
- 자체 구현 C# 정규식 기반 구문 강조 컴포넌트 `<Code>` 활용.
- 키보드 방향키 내비게이션 및 인쇄용 레이아웃(booklet) 기능 지원.

### 3) 레이아웃 카탈로그 (`showcase/index.html`)
- 기술 문서 13종 레이아웃 패턴 실물 예시 및 작성 방법 안내.
- Mermaid 다이어그램 소스 문자열을 직접 내장하여 React 컴포넌트에서 실시간 렌더링.
- 각 패턴에 해당하는 AI 프롬프트(한글) 복사 버튼 기능 제공.

## 5. 관통 예제: 크래프팅 시스템
